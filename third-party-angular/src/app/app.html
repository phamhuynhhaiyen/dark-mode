Adapter + wrapper idea

Material is themed through its own component system. Chart.js is wrapped and updated with a runtime adapter that reads CSS tokens.

ThemeService
setTheme(mode: 'light' | 'dark' | 'system') {
  localStorage.setItem('theme-mode', mode);
  this.mode.set(mode);
  this.applyTheme(mode);
}

Legacy approach

Dark mode bằng cách CSS override từng selector

Dashboard card

Light mode được viết trước như style mặc định. Khi bật dark mode, JavaScript thêm class dark vào thẻ html, sau đó CSS phía dưới override lại màu cho từng phần tử.

Pattern

/* Light mode */
.card {
  background: #ffffff;
  color: #111827;
}

/* Dark mode override */
.dark .card {
  background: #1e293b;
  color: #f8fafc;
}
